Vagrant搭建虚拟化开发环境

Vagrant 简介

Vagrant 是一款用来构建虚拟开发环境的工具，非常适合 php/python/ruby/java 这类语言开发 web 应用，“代码在我机子上运行没有问题”这种说辞将成为历史。

1、域名与服务器购买

2014年12月，在万网上购买了ursb.me这个非常魔性的域名。之后在淘宝买了个虚拟空间，将DNS解析到这个虚拟空间的IP地址。

淘宝上购买虚拟空间价格适中，但是质量无法保证，有一次帮舍友挂了学生会的投票网站，然后pv超过了2000，就把我服务器封停了，说我刷流量……

现在的话，推荐使用新浪的SAE，它也内置Wordpress，很方便的一键安装，不过貌似限制了模板和插件的使用。（云豆又涨价了！！）
