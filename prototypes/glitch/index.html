// final blend

A + E COMBINED (FINAL)

合并方案：A 每 60–120s 罕见 glitch + E 停留 220s 后启动 ambient 重新解密 + 滚动 >1500 px/s 立刻爆一次。默认 FAST 模式方便预览，可切到 REAL 看真实节奏。
A
// chromatic burst

GLITCH

偶发性的 RGB 色错位 + 像素错切，整页"卡一帧"。每 3–7 秒来一发。Cyberpunk 2077 / Watch Dogs 风。
B
// constant tremor

JITTER

整页持续高频微抖动（亚像素到 1px），永不停歇的电流嗡嗡感。Mr. Robot 终端 / CRT 显示器风。
C
// scroll datamosh

TEARING

滚动时画面像视频被压坏，水平条带错位、湍流变形；停下就缓慢恢复。Vaporwave / datamoshing 风。
D
// CRT signal

常驻扫描线 + 缓慢滚动亮带 + 偶发亮度跳变。老式赛博显示器氛围。Alien 终端 / Pip-Boy 风。
