# Grove 3D · Avatar GLB prompts

Generated by SHU-736 brainstorm — text-to-3D prompts for the 11 player
species + Mochi NPC, tuned for **Meshy 4 / Tripo 2.5 / Rodin Gen-1.5**.

## Visual north star

All 12 characters share **one consistent style** — that's what makes
the grove feel coherent. Style anchor: think *Heap Plaza × Studio Ghibli ×
Animal Crossing*. Specifically:

- **Stylized cute proportions**: big head (~40% of body), short stubby
  limbs, soft round silhouettes
- **Flat-ish soft-shaded materials** — no PBR metalness, low roughness
  variation, slightly painterly
- **Pastel-warm palette**: cream / clay / muted earth, not saturated
- **Hand-crafted feel** — light asymmetry is welcome, no overly geometric
  perfection
- **No realistic fur** (kills the style); instead clean smooth surfaces
  with a hint of paint-stroke shading

## Common prompt boilerplate (paste at the top of every prompt)

```
Stylized cute animal character, Studio Ghibli × Animal Crossing aesthetic,
big rounded head, short stubby limbs, soft warm pastel palette, gentle
soft-shaded materials with light painterly feel, no metalness, no realistic
fur, hand-crafted character design. A-pose, standing upright on two legs OR
on all fours as natural for the species, feet flat on the ground (origin at
feet), facing +Z (forward). Total height approximately 1.8 units. Auto-rigged
with idle + walk animation clips. Target ~15,000 triangles. Output: GLB with
embedded textures.
```

## Workflow

1. Pick one of:
   - **Meshy** (`meshy.ai`) — best quality + best auto-rig for cute
     characters as of 2026-05. 50 free credits/mo.
   - **Tripo** (`tripo3d.ai`) — comparable, sometimes better silhouette.
   - **Rodin** (`hyperhuman.deemos.com`) — paid but production quality.
2. Paste a prompt above. Generate. Iterate on the seed 2-3 times if
   the first one is off-style.
3. **Download as GLB** (with rig + animations if available).
4. Rename to match the table in `public/grove3d/models/README.md`:
   - `bear.glb`, `cat.glb`, etc.
5. Drop into `public/grove3d/models/`.
6. `npm run build && git commit && git push` — deploy picks it up.
7. Refresh `/nook/inner/mochi-grove/` — your shiny new model appears.

## Sanity checks before committing a GLB

- Open in `glb.app` or `gltf-viewer.donmccurdy.com` first
- Confirm scale (≈ 1.8 units tall)
- Confirm feet at y=0 (else avatar floats / sinks into ground)
- Confirm facing +Z (else avatar walks sideways)
- File size ideally < 4 MB — anything larger, consider Draco compression
- Embedded textures (single `.glb`, not `.gltf+bin+textures` triple)

## Style consistency tips

Generate all 12 in **one sitting** so the visual seed is similar across
species. If you mix sessions / weeks, models tend to drift in style.
