除夕将至，简单回顾一下我的 2019 吧。

## 学业

2019 年初一直在家写论文，论文题目叫《<管子>心论论证研究》，写起来得心应手，可能我真的比较适合做研究？

![](https://airing.ursb.me/media/15791893946694/15793595676620.jpg)


写完论文之后我还顺便写了一篇论文写作方法总结——[《文献阅读与论文写作的方法之己见》](https://me.ursb.me/archives/160.html)，这篇文章还被“共青团中央”私信请求授权转载。但是我不知道后来到底转载了没有…

![](https://airing.ursb.me/media/15791893946694/15793590293394.jpg)


写完论文之后，手还比较热，还写了一篇[《人工意识何以可能？》](https://me.ursb.me/archives/162.html)的哲学小文，想起来，当初学哲学还是非常非常有意思的。不禁想夸奖自己研究生专业的选择非常正确。

研究生生涯短短两年，转瞬即逝，而中大悠悠之学堂、僻静之校道仍历历在目。

## 工作

![](https://airing.ursb.me/media/15791893946694/15793591440899.jpg)


毕业后入职这半年来做了一些项目和许多活动，代表性的有：

* Q 音宝贝：从第一行代码一直跟进到1.0版本发布，真的待它如宝贝。

虽然很累，但是在业务的磨练下，技术确实精进了许多。

还在团队和业务线内做了两个分享：

* 《Q 音宝贝的 Hippy 端开发分享》
* 《React 16：Fiber，Hooks，Concurrent》

这一年抽空阅读 React 源码，对 React 和 Hippy 的源码都有一定的心得，除了在内部发了一些 Hippy 的经验之外，还写了一系列的 React 源码解析文章，以及年初写了一些 Web 安全系列的文章，总计 10 余篇，算下来平均月更。

![](https://airing.ursb.me/media/15447541862261/v2-64560284aca2854743af5dd5cb634376_hd.jpg-yasuo.jpg)
