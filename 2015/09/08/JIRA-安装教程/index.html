JIRA 安装教程

  • Java
技术

JIRA简介

JIRA是Atlassian公司出品的项目与事务跟踪工具，被广泛应用于缺陷跟踪、客户服务、需求收集、流程审批、任务跟踪、项目跟踪和敏捷管理等工作领域。

它比现在市场上的任何一款项目管理工具都要强大（当然，配合上收费插件和该公司旗下的其他工具之后会更加强大，适合管理100人以上的企业团队）。JIRA需要部署在自己的服务器里，安全可靠。

安装流程

目前JIRA支持Windows、Mac OS与Linux，本次安装使用的操作系统为Windows Server 2008R2 64bit，JIRA版本为6.4.7-64位版。授权从CSDN弄得，10人标准版（所以需要破解版的同学，破解方法可以自行百度）。需要先去Atlassian的官网，注册账号（注意不要使用163邮箱，会导致注册失败、授权失败），然后下载。

下载分为离线下载和在线下载，很不幸，这两个链接都需要翻墙才可以下，而且很容易发生中断意外……这里的最新版网上目前找不到，我从CSDN客服那里弄得，需要的同学可以私下找我要，有免费试用30天。

接下来看看安装步骤。

1、打开安装包

2、下一步

3、如果8080端口没有占用，选择默认，占用了选择第二个然后自行设置端口（注意服务器防火墙安全策略）

4、下一步

5、启动

6、如果没有启动，去安装界面开启服务（start_service.bat），然后启动（Launch JIRA in browser.html）

7、如果账号有授权，直接点第一步在线安装，之后就不用看了

8、如果只是想试用或者有离线的license，返回，选第二个

9、这里选择第二个，使用自己的数据库（连接速度快），这里我使用的是MySQL

10、填好数据，next

11、发现错误，缺少Mysql驱动jar包

12、这里找到我准备好的jar包，然后复制进JIRA安装目录里的lib目录下和其安装页面的lib目录下，重启服务，重新安装

13、这次成功了，直接点下一步，大功告成
