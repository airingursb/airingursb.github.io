emacs 常用命令

  • Linux
技术

昨天看了一款键盘HHKB（Happy Hacking Keyboard），售价2000左右的一款极客电容键盘，心动了……得知其是linux程序员的神奇而不适合咱们使用IDE开发的程序员……心累了。

之后了解了一下，此款键盘只所以是linux、Unix开发人员的神奇，是因为其布局可快捷键完美适配Vim和Emacs！带着一颗好奇心（闲心……），我花了一天的时间了解了一个Emacs。如果我没有Cherry青轴，没有WebStorm，没有MyEclipse，我一定会做一名linux程序员，挥舞着HHKB，在Emacs上尽情书写……

曾在黑客界流传着这样一句话，世界上只有三款编辑器，Emacs、vi和其他编辑器。这句话虽然有点夸张，不过突出了Emacs比vim还高的地位（使用Vim的童鞋不要打我）。Emacs门槛较高，没有个一年半载，很难精通甚是是熟练使用Emacs。不过今天初尝了一下HTML Mode，发现其确实非常强大，爱不释手。不过，我不是这行的……作为一个屌丝Java程序员，其实挺羡慕高大上的linux程序员。

接下来，放上我今天收集的Emacs常用命令。

  • C = Control
  • M = Meta = Alt | Esc
  • Del = Backspace
