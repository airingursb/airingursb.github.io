高德之地图组件

  • 第三方
技术

有整整十天没有写博客了，今天回来就是为了记录点简单实用的东西——高德LBS之地图组件。

地图组件简介

  • 组件特性：HTML5，URL，可视化
  • 应用范围：Native APP，Web APP
  • 功能：位置标注，路线规划，周边搜索。

文章的最后列出了一些重要的链接，大家也可以略过后文，直接到最后点链接去官网查看也可。

位置标注API

官方链接：http://lbs.amap.com/api/lightmap/guide-2/point

API：

http://m.amap.com/navi/?dest=parameter1&destName=parameter2&hideRouteIcon=1&key=parameter3

实例：

http://m.amap.com/navi/?dest=116.470098,39.992838&destName=阜通西&hideRouteIcon=1&key=(您申请的key)

参数说明：

参数名 参考值 参数说明
dest 116.470098,39.992838 (必填)用于表示待标注点的坐标。格式为（经纬，纬度），小数点后不超过6位。可通过高德坐标拾取工具获得某点的精确经纬度。
destName 阜通西 (必填)用于表示待标注点的文字描述，如“方恒国际中心”。分行显示，可使用双竖线进行强制换行
hideRouteIcon 1 (选填)只要使用该参数，就会隐藏路线功能。如果需要路线指引，不选择此参数即可。
key （您的KEY） (必填)用户通过开发者控制台获取的密钥

这里需要强调一下，高德的经纬度用英文逗号隔开，无空格，其后均有六位小数。

路线规划API

官方链接：http://lbs.amap.com/api/lightmap/guide-2/line

调用方法1：

http://m.amap.com/navi/?start=parameter1&dest=parameter2&destName=parameter3&key=parameter5

参数说明：

参数名 参考值 参数说明
start 116.403124,39.940693 (必填)起点坐标，若此参数为空，则默认使用HTML5获取用户位置作为起点。
dest 116.470098,39.992838 (必填)终点坐标。
destName 阜通西 (必填)终点名。
key （您的KEY） (必填)用户通过开发者控制台获取的密钥

####调用方法2：

http://m.amap.com/navi/?start=parameter1&dest=parameter2&destName=parameter3&naviBy=parameter4&key=parameter5

参数说明：

参数名 参考值 参数说明
start 116.403124,39.940693 (必填)起点坐标，若此参数为空，则默认使用HTML5获取用户位置作为起点。
dest 116.470098,39.992838 (必填)终点坐标。
destName 阜通西 (必填)终点名。
naviBy car (选填)表示用户希望指定的路线规划方式，取值为car/bus/walk，分别对应“驾车/公交/步行”三种方式。若用户使用此参数，则start不能为空。
key （您的KEY） (必填)用户通过开发者控制台获取的密钥

周边搜索API

官方链接：http://lbs.amap.com/api/lightmap/guide-2/localsearch

API：

http://m.amap.com/around/?locations=paratmeter1&keywords=parameters2&defaultIndex=1&defaultView=parameter3&searchRadius=parameters4&key=parameters=5

参数说明：

参数名 参考值 参数说明
locations 116.403124,39.940693 (必填)用于表示待标注点的坐标。
keywords 美食,KTV,地铁站,公交站 (必填)表示搜索的关键词，支持1—5个关键词，用英文逗号分隔。
defaultIndex 3 (必填)表示进入结果页时默认展现搜索的关键词。defaultIndex取值不能超过关键词个数。起始值为1。
defaultView map (选填)表示默认展现的视图，支持map/list两种模式，若该参数为空，则默认为list(列表视图)。
searchRadius 5000 (选填)表示搜索的范围半径，单位为：米。取值范围（1,50000），为空则默认为3000。
key （您的KEY） (必填)用户通过开发者控制台获取的密钥

推荐资源：

  • 高德官网：http://lbs.amap.com
  • 组件产品：http://lbs.amap.com/api/lightmap/guide-2/summary
  • Key申请：http://lbs.amap.com/console/key
  • 坐标拾取工具：http://lbs.amap.com/console/show/picker
